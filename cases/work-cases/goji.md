# Брендинг, сайт и трекер для студии веб-разработки

image: "/images/cases/site.png"

## Было сделано:

Начали с ресерча (конкурентный анализ, портреты ца, основные флоу, сценарии взаимодействия и пр.). На основе данных сформировал несколько визуальных бордов (как в BBE учили), выбрали направление, собрал основную страницу с адаптивами для различных экранов, ресурсов на сайт не особо много, поэтому делал на сетке 1280, чтобы не делать доп адаптив. Мобильную версию немного сплитуем с десктопом для лучшего юзабилити. Сейчас готовимся к выкатке этой новой концепции, на основе метрик планируется итерационное улучшение.
